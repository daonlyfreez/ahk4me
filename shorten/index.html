In that box above you can put a link that belongs to http://autohotkey.com or http://autohotkey.net. And you will get a shortened link, for example http://ahk4.me/qdAm8H.

If you want to link to a page in the documentation you can, for example, type "docs Hotkeys." If it's a command or function you can type "cmd MsgBox." (case sensitive)
